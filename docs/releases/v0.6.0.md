# megaton-app v0.6.0

Release date: 2026-03-06

## Highlights

- Added shared Adobe IMS OAuth client (`AdobeOAuthClient`) used by AA, Adobe Tags (Reactor), and Adobe Target integrations.
- Added Adobe Target Recommendations provider modules:
  - API client with retry/pagination/auth refresh
  - Recommendations export/apply utilities
  - Feeds export with sensitive-field redaction
  - getOffer scope detection and scoped export
- Extended Adobe Tags provider to support OAuth config (`oauth` field, config-loader parsing).
- Improved Streamlit table experience:
  - Extracted formatting/datetime detection into `app/ui/table_format.py`
  - Added display controls (date format, thousands separators, decimals)
  - Added `column_types` hints support in schema/validator/query builder

## Compatibility Notes

- No breaking changes in CLI parameter schema.
- Existing Adobe Tags bearer-token configuration remains supported.
- OAuth can be enabled gradually per project/config.

- Full test suite passed: `489 passed`.
